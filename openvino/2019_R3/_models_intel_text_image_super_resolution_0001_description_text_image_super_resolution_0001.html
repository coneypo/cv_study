text-image-super-resolution-0001

Use Case and High-Level Description

A tiny model to upscale scanned images with text. The model used the ConvTranspose2d layer instead of the PixelShuffle layer. As a result, the model can be launched on GPU and MYRIAD devices and support the reshape function.

Inputs

  1. name: 0, shape: [1x3x768x1280] - An input image in the format [BxCxHxW], where:
    • B - batch size
    • C - number of channels
    • H - image height
    • W - image width

Outputs

One blob with shapes [1, 1, 1080, 1920]. The blob contains an image after super resolution.
