OpenVINO Toolkit
Install DL Workbench from the Intel® Distribution for OpenVINO™ Toolkit Package

You can install DL Workbench by installing the OpenVINO™ package that contains the tool.

Installation Prerequisites

You can download the Intel® Distribution of OpenVINO™ toolkit for Linux from the download page (registration required).

Follow the instructions to Install the Intel® Distribution of OpenVINO™ toolkit for Linux*.

NOTE: Do not remove the archive you used to install the OpenVINO™ package, because the archive is required for running DL Workbench.

Proxy Settings

If you are behind a corporate proxy, before running DL Workbench you need to run the following commands to set environment variables:

export https_proxy=<https_proxy>
export http_proxy=<http_proxy>
export no_proxy=<no_proxy>

Run DL Workbench

Open a terminal in the DL Workbench folder. The path to the folder is /<path_to_installed_package>/deployment_tools/tools/workbench.

Run the following command, specifying the path to the downloaded archive with the OpenVINO™ package:

./run_openvino_workbench.sh -PACKAGE_PATH <path-to-archive>

This command sets up the environment for a DL Workbench Docker* image, builds the Docker image, and runs the Docker container with DL Workbench. Once the command executes, point your browser to https://127.0.0.1:5665. The DL Workbench Get Started page appears:

You have successfully installed OpenVINO™ DL Workbench. Now you can move on to Work with Models and Sample Datasets.
