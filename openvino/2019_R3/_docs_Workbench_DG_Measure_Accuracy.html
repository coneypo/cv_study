Measure Accuracy for the First Time

NOTE: Accuracy measurements are not available for projects that use generic models, OpenVINO™ models, or generated datasets.

Once you select a model and a dataset, and run a baseline inference, the Projects page appears. To measure accuracy of a model, click on the rotating arrows sign:

Note: Accuracy is not available if you use autogenerated datasets.

The measurement process takes a few seconds, and the accuracy number appears:

Remeasure Accuracy

A relatively low accuracy estimation can be due to incorrectly selected options in the Advanced Configuration form in the Select Model step.

To remeasure accuracy, return to the Configuration menu by clicking on the gear sign in the Accuracy column and fill out the form again, then restart the accuracy measurement.

The process takes a few seconds, and the accuracy number appears.
