Install DL Workbench

Prerequisites

OpenVINO™ DL Workbench requires additional software to be installed to satisfy general OpenVINO™ requirements for Linux* systems.

Prerequisite Minimum Requirement
Operating SystemUbuntu* 16.04, Ubuntu 18.04. Other Linux distributions, such as CentOS* 7, are not validated.
Windows* can be used to run DL Workbench from Docker Hub*.
Available Storage Space 5 GB
Docker*Docker CE 18.06.1
Web BrowserGoogle Chrome* 72
Other browsers, like Mozilla Firefox* 65 or Apple Safari* 12, are not validated. Microsoft Internet Explorer* is not recommended.

Installation Methods

To install the OpenVINO™ DL Workbench tool, choose one of the following methods:

  • Install from Docker Hub* (Includes installing on Linux and Windows)
  • Install from the Intel® Distribution of OpenVINO™ toolkit package
  • Install from Dockerfile

For additional options, such as launching the DL Workbench container and restarting the container, see Advanced Topics.
